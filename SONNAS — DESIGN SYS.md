# Sonnas & CAFE — DESIGN SYSTEM

## 1. BRAND DIRECTION

Style: Luxury • Soft • Feminine • Minimal • Premium
Inspiration: French patisserie aesthetic with warm emotional appeal
Primary Focus: Product imagery + elegant UI

---

## 2. COLOR SYSTEM (PINK-DOMINANT)

### Primary Colors

* Blush Pink → #F4C2C2 (main brand identity)
* Soft Rose → #E8A9A9 (buttons, highlights)
* Deep Rose → #D88C8C (hover states)

### Neutral Base (Luxury Feel)

* Cream White → #F7F3EF (main background)
* Warm Beige → #E8DED4 (sections/cards)
* Soft Ivory → #FFF9F7 (alternate background)

### Text Colors

* Primary Text → #2B2B2B
* Secondary Text → #6E6E6E
* Muted Text → #9A9A9A

### Accent Colors

* Gold Accent → #C9A27E (premium feel)
* Cocoa Brown → #5A3E36 (depth contrast)

---

## 3. COLOR USAGE RULES

* Background = always light (cream / ivory)
* Pink = used for highlights, CTAs, branding
* Gold = very minimal (premium accent only)
* Never use harsh or saturated colors
* Maintain soft contrast, avoid pure black/white

---

## 4. TYPOGRAPHY

### Headings

Font: Playfair Display
Style: Elegant serif
Usage: Titles, hero text

### Body Text

Font: Inter
Style: Clean sans-serif

### Rules

* Large headings (32px–64px)
* Thin to medium weight
* Increased letter spacing for elegance

---

## 5. LAYOUT SYSTEM

### General Principles

* Maximum whitespace
* Centered layouts
* Grid-based product display
* No clutter

### Section Structure

1. Hero (large image + minimal text)
2. Featured products (grid)
3. Categories
4. Story/About
5. Footer

---

## 6. UI COMPONENT STYLES

### Buttons

* Background: Soft Rose (#E8A9A9)
* Text: White
* Border radius: 12px
* Hover: Darker rose (#D88C8C)

### Cards (Product)

* Background: White / Ivory
* Shadow: Soft, subtle
* Rounded corners: 16px
* Minimal text

### Navbar

* Transparent or cream background
* Thin typography
* Minimal links

---

## 7. PRODUCT DISPLAY RULES

* Large, high-quality images
* No cluttered labels
* Focus on visual appeal
* Name + price only

---

## 8. INTERACTION STYLE

* Smooth transitions (0.3s ease)
* Hover scale on images
* Fade-in animations
* No aggressive motion

---

## 9. BRAND FEEL CHECKLIST

Every screen must feel:

* Soft
* Premium
* Clean
* Emotional
* Dessert-focused

If it feels crowded → remove elements
If it feels plain → improve imagery, not UI

---

## 10. DO NOT DO

* No bright neon pink
* No heavy shadows
* No crowded layouts
* No too many fonts
* No sharp edges
